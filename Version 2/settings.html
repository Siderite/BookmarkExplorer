  • Assuming the current browsed page is bookmarked, you can navigate to the next or the previous URL from the same bookmark folder
  • You can use context menus, keyboard shortcuts or the buttons in the extension popup to navigate
  • You can open a management page for the current bookmark folder, allowing you to select and remove read URLs
  • In case you delete bookmarks by mistake, there is a deleted items page that allows you to restore what was deleted
  • The Read Later functionality allows you to directly bookmark a link, without keeping its page open
  • The extension allows for importing/exporting of links as a simple list of URLs
  • The extension does not connect to outside servers, everything is stored locally (warning: uninstalling it loses the deleted bookmarks data)
  • You can tweak the way URLs are compared to each other in the Advanced section, for specific sites
